toolbox.css
===========

Instead of creating new classes for each element in development, let toolbox.css do the heavy lifting for you with helper classes.

display
-------
```
.block {
  display: block;
}

.inline {
  display: inline;
}

.inline-block {
  display: inline-block; 
}
```


border-box
----------
```
.border-box {
  box-sizing: border-box;
}
```

float
-----

```
.pull-left {
  float: left;
}

.pull-right {
  float: right;
}
```

width
-----
```
.full-width {
	width: 100%;
}
```
